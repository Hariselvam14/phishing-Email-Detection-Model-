# Phishing Email Detection Model

## Description
This project is a simple machine learning model developed using Python and Scikit-learn to detect phishing emails.

The model analyzes email text and classifies it as either **Phishing** or **Safe** based on keywords and patterns.

## Features
- Detects phishing emails using machine learning
- Uses text feature extraction with CountVectorizer
- Classifies emails as "Phishing" or "Safe"
- Displays model accuracy
- Shows confusion matrix for performance evaluation

## Technologies Used
- Python
- Scikit-learn
- CountVectorizer
- Multinomial Naive Bayes
